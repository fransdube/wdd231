Website Plan

Site Name

Johanesburg eBiking Club

This name represents a club focused on electric biking in the Johanesburg area.

Site Purpose

The site provides a hub for regional e-biking by listing trail and e-bike recommendations, along with a membership information form.

Scenarios

  • What is the best e-bike for mountainous areas?
  • Where can I find contact information for the club's directors?

Color Schema

Primary Color: #2A9D8F (used for headings and accents)

Secondary Color: #264653 (used for background and paragraphs)

Typography

Heading Font: "Roboto", sans-serif

Body Font: "Open Sans", sans-serif
